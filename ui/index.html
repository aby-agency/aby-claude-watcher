About Aby Claude Watcher

Real-time dashboard for your Claude Code sessions.

  • Auto-detects every Claude Code session as it spawns
  • Live state per session: thinking, running, waiting, completed
  • Notifies you when Claude is waiting for your input
  • One click to focus the originating terminal, or to resume
  • Tracks 5h and 7-day usage windows with countdown to reset
